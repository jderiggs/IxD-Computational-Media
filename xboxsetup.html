Xbox Adaptive Controller Setup

After plugging in the Xbox Adaptive Controller, You should download the Xbox Accessories App off of the Microsoft Store in order to create custom button programs for the controller. You can have multiple profiles, with the position of the LED on the controller denoting which profile is active at any given time.
