Overlays

OVR Advanced Settings

OVR Advanced settings allows fine adjustment of settings such as playspace height, and room borders.

fpsVR

fpsVR allows users to monitor CPU and GPU performance during games to identify bottlenecks. It also allows tracking of controller/headset battery.

OVR Toolkit/XSOverlay

Both of these programs function similarly, they allow windows to be opened and interacted with while in a VR experience seamlessly.

System refers to the platform that your headset supports. For example, the Meta Quest 2 uses Oculus, but can use SteamVR if linked to a computer.
